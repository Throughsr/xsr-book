# text

## 文本溢出
```css
.x-text-truncate {
    overflow:hidden;
    text-overflow:ellipsis;
    white-space:nowrap;
}
```

## 文本对齐
```css
.x-text-left {
    text-align:left;
}
.x-text-right {
    text-align:right;
}
.x-text-center {
    text-align:center;
}
```

## 文字瘦高
```css
.x-text-tall { 
    display: inline-block;transform:scale(1,1.5);
}
```
