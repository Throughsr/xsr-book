```css
.x-whitespace.x-whitespace-xs {
  height: 3px;
}

.x-whitespace.x-whitespace-sm {
  height: 6px;
}

.x-whitespace.x-whitespace-md {
  height: 9px;
}

.x-whitespace.x-whitespace-lg {
  height: 15px;
}

.x-whitespace.x-whitespace-xl {
  height: 21px;
}
```
