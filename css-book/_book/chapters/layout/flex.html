flex 水平滑动

适用于图片水平列表滑动

.level-flex-box {
  display: flex;
  width: 100%;
  padding: 0 10px;
  box-sizing: border;
  overflow-x: scroll;
  -webkit-overflow-scrolling: touch;
}

.level-flex-item {
  flex-shrink: 0;
  margin-right: 10px;
  width: 100px;
}

flex 属性

属性 描述
flex-grow 扩展比率，父元素多出来的处理方法
flex-shrink 收缩比率，父元素不足的处理方法
flex-basis width的替代品
